**Real-Time Market Data Streaming Pipeline**


I developed this project as a hands-on learning exercise to strengthen my data engineering skills and gain practical experience with modern data pipeline tools.

This project demonstrates an end-to-end real-time data engineering pipeline that streams gold price data and foreign exchange (FX) rates, processes the data using Apache Kafka, stores it in Snowflake, transforms it using dbt, and visualizes the results through an interactive Power BI dashboard.

The objective of this project is to simulate a modern financial data analytics pipeline that ingests live market data, processes it in near real-time, and makes it available for analysis and reporting.

The pipeline continuously processes gold price updates along with currency exchange rates, converts the prices into multiple currencies and weight units, and presents the results in an interactive dashboard that allows users to explore price movements over time.

Please refer to Project Architecture for details and data lineage on Source, Transformation, target and analytical layers.
